Discussing Map Metrics


	March 3, 2013.
	Apparently CM gives overincreased results for some Multiban Maps.
	The human solution for Erim Sever, All Levels, Map 361
	has p,i,t = 505.129.19 and gives
	Creativity Metric, C = d/v ~ 83
	which is apparently too high.
	Indeed, the solution is almost straightforward and purely "geometrical", but
	it equals this map to some possibly smaller, but harder maps.
