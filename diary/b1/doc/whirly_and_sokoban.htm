	As of today, January 3, 2013, roughly:

	there are push, pull, jump, swap, pass, block, leap ...          7 interactions implemented ...
	there are: hero, box, target, htarget, wall, ground ...          6 races ...
	10 colors ... so 60 breeds ... 20 dynamic breeds ...             20 x 60 cells
	in interaction matrix = 1200 and
	each cell can take independently 7 interactions ... so ...       W = 71200 interaction matrices or games is possible ...

	( More than 1080, possible number of atoms in Observable Universe. )

	Sokoban is one of these games. Sokoban Gameland is full of famous maps, web sites, fans, and developments.
	( This site claims having 22700 levels ... for Sokoban. )

	Each other "W-game" is a subtle or strong deviation from Sokoban cannonical interaction matrix and possibly stems
	an independent Gameland which is parallel to Sokoban.

	Perhaps, 99.99% of these W-games can be just a junk or too complex to be interesting for a human, but imagine 
	each significant Gameland has own fans, depository of maps, and of course, gameland.group.yahoo.com ...

 

	While yahoo.com does not have enough hard drive space for new 71200 groups, we show only a few Sandbox games below.
	The reader may evaluate can these games grow up from sandbox to a Gameland.
	We don't know yet ...
