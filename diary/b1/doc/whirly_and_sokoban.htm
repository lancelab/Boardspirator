	As of today, January 3, 2013, roughly:

	there are push, pull, jump, swap, pass, block, leap ...          7 interactions implemented ...
	there are: hero, box, target, htarget, wall, ground ...          6 races ...
	10 colors ... so 60 breeds ... 20 dynamic breeds ...             20 x 60 cells
	in interaction matrix = 1200 and
	each cell can take independently 7 interactions ... so ...       G = 7 ^ 1200 interaction matrices or games
	is possible ...
	More than 10 ^ 80, possible number of atoms in Observable
	Universe. 

	Each game is a subtle or strong deviation from Sokoban cannonical
	interaction matrix and possibly stems an independent Manifold of
	gameplays which is parallel to Sokoban Manifold.

	Perhaps, 99% of these G-games can be just a junk or too complex
	to be interesting for a human ... 

	Each Manifold will have own fans, depository of maps (The site
	claims having 22700 levels ... for Sokoban Manifold ), and of course, manifold.group.yahoo.com.

	While yahoo.com does not have enough space for new groups, we show only a few
	games in the table below. It is up to the reader to decide are these games rubbish or
	stem a glorious Manifold. We don't know yet ...

   If box or hero are colorless, rules are the same.
   Otherwise:
